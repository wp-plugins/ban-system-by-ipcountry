This plugin helps you to block unwanted visitors by their ip, or contry.

== Description ==

You can ban users by IP(for a certain period), and you can also ban visitors from certain countries.

Main Features:

*   Ban users by IP or by Country
*   Redirect or custom message for banned users
*   Cookie banning for security
*   Ban reason and time/period can be set up
*   Unban option
*   Lastest statistics - Logging every visitor for a 2 day period

== Installation ==

As easy as a piece of cake!

1. Install and activate the Plugin from Dashboard > Plugins

Or, from FTP/Upload
1. Download the ban.zip file
2. Upload it from Dashboard > Plugins > Add New > Upload or upload it in: blog.com/wp-content/plugins/
3. Go to Dashboard > Plugins and Activate it
